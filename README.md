
# SBWeb
A website designed for swahilibox community

Make sure you have ruby and rails installed.

To get your local
 
	 git clone `git@github.com:RailsGirlsMsa/SBWeb.git`

	 cd SBWeb

	 run bundle install

	 run rails server or rails s
